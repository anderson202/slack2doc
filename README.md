## Setup

### Environment Variables
```
export SLACK_BOT_TOKEN=<token>
export SLACK_SIGNING_SECRET=<token>
```

### Install requirements
```
pip install -r requirements.txt
```

#### ngrok

https://ngrok.com/download


```
brew install ngrok/ngrok/ngrok
```


#### nodemon

```
npm i -g nodemon
```


### Run Service
```
./start.sh
```
